* Linear Algebra & Calculus

The rotation matrix $R(\theta)$:
\[ R(\theta) = \begin{pmatrix}
\cos\theta & -\sin\theta \\
\sin\theta & \cos\theta
\end{pmatrix} \]

The absolute value function is defined as:
\[ |x| = \begin{cases}
 x & \text{if } x \ge 0 \\
